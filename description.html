Introduction

This is a simple project that demonstrates how to use a Joystick control in UWP apps as well as some insight into it’s working. Here’s a sample preview of the control we’re talking about.

Pre-Requisites

To build this project, you need Visual Studio 2015 with UWP SDK installed on your machine i.e. Check the Universal Windows App Development Tools option when installing Visual Studio.

If you have Visual Studio already installed, then just run the setup and check Universal Windows App Development Tools option. It will only install the required components.

 

See it in Action

1)      Download and extract the .zip file.

2)      Open the .sln file in Visual Studio.

3)      Run the application

Building the project automatically restores any required packages but if it doesn’t then you can restore it manually, right click on the solution in solution explorer then click Restore NuGet Packages.

 

Use it in Your Own App


Step 1: Add JoystickUserControl

Method 1: Copy UserControl Code

You should use method to learn about the working of Joystick UserControl or if you want to modify and optimize the control according to your need.

  - Drag and drop the three files from the folder JoystickUserControl (namely Joystick.xaml, Joystick.xaml.cs and JoystickEventArgs.cs) into your UWP project inside Visual Studio.

Method 2: Installing Nuget Package

A cleaner way to get Joystick control in your app is through NuGet.

  - Go to NuGet Package Manager Console and enter the following command

Install-package JoystickUserControl

Then build the project.

 

Step 2: Add Namespace 

Add the xmlns:JoystickControl="using:JoystickUserControl"  namespace to your mainpage.xaml or whatever page you want to add joystick to.

 

Step 3: Add the Joystick control to your page.

Add Joystick control using the following statement,

Step 4: Customize Joystick Control

You can change a lot of properties but let’s just run the app to see how our Joystick looks like with default settings.

Properties

Name

Description

InnerDiameter

Sets the diameter of inner circle.

Default: 60px

InnerFill

Color of inner circle. It overrides theme style. It overrides the respective color set by theme. You can use it to tweak theme style.

InnerStroke

Border color of inner circle. It overrides the respective color set by theme. You can use it to tweak theme style.

InnerStrokeDashArray

This property sets the length of stroke and gaps to create dash pattern of inner circle.

InnerStrokeDashCap

This property sets the shape of dash edges of inner circle. Default is Flat.

InnerStrokeDashOffset

Sets the offset to start the dash pattern of inner circle. You can use it to rotate dash pattern.

InnerStrokeThickness

Sets the thickness of inner circle border.

OuterDiameter

Sets the diameter of the boundary of joystick.

Default: 100px

OuterFill

Color of outer circle. It overrides the respective color set by theme. You can use it to tweak theme style.

OuterStroke

Border color of outer circle (joystick boundary). It overrides the respective color set by theme. You can use it to tweak theme style.

OuterStrokeDashArray

This property sets the length of stroke and gaps to create dash pattern of outer circle.

OuterStrokeDashCap

This property sets the shape of dash edges of outer circle. Default is Flat.

OuterStrokeDashOffset

Sets the offset to start the dash pattern of outer circle. You can use it to rotate the dash pattern.

OuterStrokeThickness

Sets the thickness of outer circle border.

Theme

Sets the theme of Joystick.

Themes: AccentTheme, Dark and Light.

Default: AccentTheme

Note: Theme styles will be overridden in case you set the styling properties (InnerFill, InnerStroke, OuterFill, OuterStroke) explicitly.

XValue

Gets the current x-axis value of joystick between -1 to 1. Negative on left side and positive on right.

YValue

Gets the current y-axis value of joystick between -1 to 1. Negative on lower side and positive on upper side.

 

Events

Name

Description

OnJoystickMoved

This event is called whenever the joystick is moved.

OnJoystickPressed

This event is called whenever the joystick is pressed (tap, touch, mouse click).

OnJoystickReleased

This event is called whenever the joystick is released even if the pointer left the joystick boundary while pressed.

How it Works?

In a nutshell, Joystick control is simply a UserControl with two <Ellipse> controls, one inside the other. Inner ellipse acts as the controller and outer as the bounding area for joystick. Mouse events are used to perform calculations and translate the joystick control whenever an interaction happens.

XAML
Edit|Remove
xaml 
    <Grid DataContext="{Binding ElementName=JoystickControl}">
        <Ellipse x:Name="ControlArea" 
                 Height="{Binding OuterDiameter}" 
                 Width="{Binding OuterDiameter}"
                 Fill="{Binding OuterFill}"
                 Stroke="{Binding OuterStroke}"
                 StrokeThickness="{Binding OuterStrokeThickness}"
                 StrokeDashArray="{Binding OuterStrokeDashArray}"
                 StrokeDashCap="{Binding OuterStrokeDashCap}"
                 StrokeDashOffset="{Binding OuterStrokeDashOffset}"
                 />

        <Ellipse x:Name="Controller" 
                 Height="{Binding InnerDiameter}" 
                 Width="{Binding InnerDiameter}"
                 Fill="{Binding InnerFill}"
                 Stroke="{Binding InnerStroke}"
                 StrokeThickness="{Binding InnerStrokeThickness}"
                 StrokeDashArray="{Binding InnerStrokeDashArray}"
                 StrokeDashCap="{Binding InnerStrokeDashCap}"
                 StrokeDashOffset="{Binding InnerStrokeDashOffset}"
                 PointerPressed="Controller_PointerPressed">
            <Ellipse.RenderTransform>
                <TranslateTransform x:Name="JoystickTransform"></TranslateTransform>
            </Ellipse.RenderTransform>
        </Ellipse>

Databinding makes it a lot easier to keep track of changes in joystick size, theme and colors at runtime if you want to create animations.
